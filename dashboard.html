How It Works
El AirGradient ONE envía datos via webhook/API y el sensor Tuya PIR via Make.com. Los datos se escriben automáticamente en Google Sheets. Este dashboard lee la última fila de cada hoja cada 60 segundos.

Paso 1: Crear Google Sheet con columnas: timestamp, voc, co2, pm, humid, temp, nox, people
Paso 2: Configurar Make.com para escribir datos de AirGradient API + Tuya PIR en la hoja
Paso 3: Publicar la hoja y pegar el Sheet ID abajo
Paso 4: Cambiar a Modo Live — el dashboard actualiza solo cada minuto
